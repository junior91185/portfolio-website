IAM Application Project

Technologies: Linux (Ubuntu), Bash

Project Overview

During my time as Identity Aceess Management (IAM) Intern at the University of Rochester Information Security Departemnt, I was tasked with conducting a compliance auduit for 30+ IAM applications . Applications included Duo Two Factor Authentication, CyberArk, Guest Account System, and more. One of the many taks to complete for the compliance auduit were to verify the accuracy of critical attributes (service components, host details, IP addresses) on the Info Secs server page. I created a simple to automate DNS validation, replacing manual nslookup checks and ensuring accurate IP-to-hostname mappings.


Purpose

Create a script to automate the verification of IP addresses and hostnames for accurate entry into the application catalog. Ensuring that each IP address has the correct hostname was essential for maintaining precise and reliable documentation.


Example Usage

Create a text or csv file with a list of IP addresses (one per line).

8.8.8.8
4.4.4.4
216.239.36.10

Run the script with the input file as a parameter:

./main.sh path/to/your/input_file.csv

Results

The DNS Lookup Tool produces a clear and organized output that lists each IP address alongside its corresponding hostname. This helps in quickly verifying and documenting network information.

Example Output:

8.8.8.8    dns.google
4.4.4.4    quad9.net
216.239.36.10    ns3.google.com
